# 웹 3쿠션 당구 GDD (MVP)

## 1. 문서 정보
- 제목: 웹 3쿠션 당구 게임 기획 문서
- 버전: v1.0
- 작성일: 2026-02-20
- 범위: MVP

## 2. 게임 개요
- 장르: 실시간 멀티플레이 웹 당구
- 모드: 3쿠션 단일 모드
- 핵심 루프: 방 입장 -> 턴제로 샷 진행 -> 먼저 10점 달성

## 3. 핵심 정책
- 게임 진행 중인 방에는 누구도 입장할 수 없다.
- 방당 최대 6명 플레이어이며 관전자는 없다.
- 한 플레이어가 10점에 도달하면 즉시 경기가 종료된다.

## 4. 정보 구조
1. Depth 1: 로그인
2. Depth 2: 로비
3. Depth 3: 게임방(게임 화면)

## 5. Depth 1 - 로그인
### 5.1 게스트 로그인
- 닉네임은 필수다.
- 닉네임 중복은 허용된다.

### 5.2 회원 로그인/가입
- ID/PW로 회원가입과 로그인을 수행한다.
- 이메일 인증이 없고 SMTP 의존성이 없다.

## 6. Depth 2 - 로비
### 6.1 방 목록
- 초기 방 9개를 3x3으로 표시한다.
- 무한 스크롤을 지원한다.
- 페이지네이션 단위는 로드당 9개로 고정한다.

### 6.2 정렬 우선순위
1. 대기중인 방(미시작)을 우선 노출한다.
2. 대기 인원이 적은 방을 우선한다.
3. 최근 생성된 방을 우선한다.

### 6.3 입장 규칙
- 진행 중인 방은 입장할 수 없다.
- 대기/종료 상태 방은 누구나 입장할 수 있으며 이전 강퇴 이력과 무관하다.

## 7. 방 생성 정책
- 입력값은 방 제목만 사용한다.
- 방 제목 최대 길이는 15자다.
- 금지 문자는 없다.
- 제목 중복은 허용된다.
- 비밀번호는 없다.
- 최대 인원은 6명으로 고정한다.

## 8. Depth 3 - 게임방 / 게임 화면
### 8.1 레이아웃
- Top View 당구대를 중앙에 배치한다.
- 플레이어 목록, 현재 턴, 점수판을 표시한다.
- 게임방 채팅 패널을 제공한다.

### 8.2 인원 규칙
- 최대 인원: 6명
- 게임 시작 최소 인원: 2명

### 8.3 조작 방식
#### 큐 조작 (마우스)
- 마우스 좌우 이동:
  - 큐가 수구를 중심으로 수평 360도 회전한다.
  - 마우스를 좌측으로 이동하면 큐가 시계방향으로 회전한다.
  - 마우스를 우측으로 이동하면 큐가 시계 반대방향으로 회전한다.
- 마우스 상하 이동:
  - 큐가 수구를 중심으로 수직 회전한다.
  - 마우스를 위로 올리면 큐 뒤쪽이 들리고 팁이 수구 상단을 겨냥한다.
  - 마우스를 아래로 내리면 큐 뒤쪽이 내려가고 팁이 수구 하단을 겨냥한다.
  - 큐 고각 범위: 최소 `0도`(수평), 최대 `89도`.
- 마우스 드래그 스트로크:
  - 좌클릭을 누른 상태에서 스트로크를 준비한다.
  - 아래 방향으로 드래그한다.
  - 좌클릭 버튼을 떼면 스트로크가 실행된다.
  - 드래그 범위: `10 px` ~ `400 px`.
  - `10 px`는 최소 공 속도 `1 m/s`에 대응한다.
  - `400 px`는 최대 공 속도 `50 km/h` (`13.89 m/s`)에 대응한다.
  - 범위를 벗어난 입력값은 `[10, 400]`으로 클램프한다.
  - 권장 매핑은 드래그 거리와 초기 공 속도의 선형 보간이다.

#### 스핀 조작 (키보드)
- `W`: 당점을 12시 방향으로 이동한다.
- `S`: 당점을 6시 방향으로 이동한다.
- `A`: 당점을 9시 방향으로 이동한다.
- `D`: 당점을 3시 방향으로 이동한다.

## 9. 방장 시스템
### 9.1 방장 권한
- 게임 시작
- 플레이어 강퇴
- 재경기 시작

### 9.2 방장 위임
- 방장이 나가면 입장 순서 기준 다음 플레이어에게 방장이 자동 위임된다.
- 방장이 나가도 방은 삭제되지 않는다.

### 9.3 강퇴 정책
- 투표 없이 즉시 강퇴한다.
- 강퇴된 플레이어는 방이 진행 중이 아닐 때만 재입장할 수 있다.

## 10. 경기 규칙
### 10.1 시작 규칙
- 경기는 방장 시작 버튼으로만 시작한다.
- 준비 상태 확인은 필요 없다.

### 10.2 턴 규칙
- 턴 순서는 입장 순서로 고정한다.
- 턴 제한시간은 10초다.
- 시간 초과 시 추가 패널티 없이 자동 턴 스킵한다.

### 10.3 승패 조건
- 먼저 10점을 달성한 플레이어가 승리한다.
- 10점 달성 즉시 경기가 종료된다.
- 즉시 종료 구조이므로 동점 처리 규칙은 필요 없다.

### 10.4 이탈/연결 해제
- 게임 중 이탈은 즉시 패배 처리한다.
- 1명만 남으면 경기를 즉시 종료하고 남은 플레이어가 승리한다.

### 10.5 재경기
- 재경기는 방장 버튼으로 시작한다.
- 재경기 시작 시 모든 플레이어 점수를 0으로 초기화한다.
- 턴 순서는 입장 순서를 유지한다.

## 11. 채팅
- 범위: 게임방 내부 채팅 전용
- 전송 제한: 플레이어당 3초에 1메시지
- 저장 방식: 메모리 전용이며 영속 저장하지 않는다.
- 욕설 필터는 MVP 범위에서 제공하지 않는다.

## 12. 상태 모델 (MVP)
### 12.1 방 상태
- `WAITING`: 게임 시작 전 상태
- `IN_GAME`: 게임 진행 중 상태(방 잠금)
- `FINISHED`: 게임 종료 상태, 재경기 가능

### 12.2 방 상태 전이
1. `WAITING` -> `IN_GAME` (방장이 시작을 누르고 인원이 2명 이상일 때)
2. `IN_GAME` -> `FINISHED` (어떤 플레이어가 10점 달성 또는 1명만 남았을 때)
3. `FINISHED` -> `IN_GAME` (방장이 재경기를 시작할 때)

### 12.3 플레이어 상태
- `IN_ROOM`: 방에 입장한 상태
- `PLAYING`: 경기 진행 상태
- `WIN`: 승리 상태
- `LOSE`: 패배 상태
- `KICKED`: 강퇴 상태, 방 상태에 따라 재입장 가능

## 13. 물리 입력 파라미터 (물리 명세서에서 상세화)
### 13.1 공
- 지름: 61.5 mm
- 무게: 210 g
- 공-공 반발계수: e ~= 0.92 ~ 0.98
- 재질: 페놀 수지

### 13.2 당구대
- 내경: 2.844 m x 1.422 m
- 외경: 3,100 mm x 1,700 mm
- 쿠션 높이: 0.037 m
- 공-쿠션 반발계수: e ~= 0.70 ~ 0.75
- 천 마찰계수:
  - 미끄럼 마찰계수 ~= 0.2
  - 구름 마찰계수 ~= 0.01 ~ 0.015

### 13.3 큐
- 길이: 1.41 m
- 무게: 500 g
- 팁 지름: 12 mm
- 팁-공 마찰계수: 0.7
- 팁 반발계수: 0.7
